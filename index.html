MEDREF - Hospital Laboratory Referral System

This project focuses on developing a web-based referral system for hospital laboratories, addressing the inefficiencies and potential inaccuracies of traditional paper-based methods still prevalent in many government hospitals. The primary goal is to create a digital platform that streamlines the exchange of physician-prescribed test requests and subsequent results between physicians and laboratories, thereby enhancing the accuracy and efficiency of diagnostic services.

Key Features:

  • Role-based access for doctors, lab technicians, and admins
  • Digital referral slips with test type and lab selection
  • Patient medical history and test result tracking
  • Real-time referral status updates
  • Patient search and filtering capabilities
  • Improved coordination between departments

DataLake - Video Analytics System

Developed a DataLake architecture to process and analyze camera footage for various applications, including object detection and activity recognition.

Key Contributions:

  • Contributed to developing a scalable, on-premise data lake for metadata processing and storage of camera footage.
  • Researched and applied object localization techniques using uncalibrated surveillance footage, combining YOLO and Swin Transformers for feature extraction and GPS regression.
  • Produced a custom dataset using Google Street View, annotating and geo-tagging vehicles using GPS offset calculations for accurate vehicle geo-localization.
  • Integrated with MinIO object storage and implemented a scalable data lake architecture for long-term metadata storage.

Enhancing Vehicle Localization Accuracy at Traffic Intersections

An individual research project focused on improving vehicle geo-localization accuracy in real time at traffic intersections using uncalibrated surveillance cameras.

Methodology:

  • Construct a dataset using Google Street View images with manual annotation
  • Enhance YOLOv8 with Swin Transformer, RFA, and ECA for improved feature extraction
  • Integrate GPS regression head for precise distance and azimuth estimation
  • Apply theoretical calculations for latitude and longitude offsets
  • Use ByteTrack for robust, real-time multi-object vehicle tracking
  • Compare datasets, detection methods, GPS integration, and tracking algorithms

Maze Puzzle Solver - Intelligent Pathfinding System

An interactive maze-solving system that generates square mazes and finds the shortest path from start to goal using advanced algorithms and visualizations.

Key Features:

  • Maze generation using Kruskals algorithm with a single-solution guarantee
  • Multiple solving algorithms: BFS, DFS, and A* search
  • Real-time visualization of the maze generation and solving process
  • Randomized start and goal placement on maze borders
  • Execution time display for both generation and solving
  • Optimized data structures for faster performance on larger mazes

Leave Management System - Desktop Application

A .NET Framework 4.8-based desktop application designed to manage employee leave requests efficiently. The system consists of two main applications: an Employee Application for submitting and tracking leave requests, and an Admin Application for approving/rejecting requests, managing leave balances, and defining rosters.

Key Features:

  • Employee Application: Login, submit leave requests, view leave history
  • Admin Application: Login, approve/reject leave requests
  • Manage employee leave balances and define rosters
  • SQL Server (LocalDB) backend with secure data handling
  • Windows Forms user interface for easy navigation
